# Independent Music Label Template

Echo Chamber Records is a premium, alternative, editorial HTML template designed for independent record labels, music management agencies, music collectives, and publishers.

## Key Features

- **Editorial Design System**: Customized color palettes for both dark and light modes, prioritizing typography (Bebas Neue, Space Grotesk, Inter).
- **Responsive Layout**: Designed mobile-first, supporting all modern viewports (320px up to ultra-wide displays).
- **Sticky Header & Custom Navigation**: Dropdown links, dynamic scroll shadow states, and theme switching controls.
- **Mock Audio Player & Track Scrubber**: Full client-side UI playback states for catalog listening tracks.
- **Sync & Music Licensing Hub**: Custom music supervisionBrief wizards and mood filtering grids.
- **Submissions System**: Editorial unsigned artist demo guidance and forms.
- **GSAP Animation Suite**: Fade-ups, image reveals, staggered entries, hero parallax, and rotating vinyl cover transitions.

## Setup Instructions

1. Open `index.html` in any modern web browser to view the template locally.
2. Toggle between Light Mode and Dark Mode (Primary Experience) using the sun/moon icon in the sticky navigation header.
3. Interact with the audio players to see active play/pause states and scrub timeline updates.
